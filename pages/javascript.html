Section 6.1 · Definition

What is JavaScript?

JavaScript (JS) is a high-level, interpreted, multi-paradigm programming language originally created by Brendan Eich at Netscape in 1995. It is the only programming language that runs natively in web browsers, making it indispensable for adding interactivity to web pages. Since Node.js (2009), JavaScript can also run on the server side, making it a true full stack language (Flanagan 2020, p. 1).

JavaScript is the third core technology of the web alongside HTML and CSS: HTML provides structure, CSS provides presentation, and JavaScript provides behaviour and interactivity (MDN 2024). Modern ES6+ introduced arrow functions, template literals, destructuring, async/await, and the Promises API.

// Arrow function with template literal
const createCard = (p) => {
  const price = p.type === 'rent' ? `$${p.price}/wk` : `$${p.price.toLocaleString()}`;
  return `<div class="card"><h5>${p.title}</h5><p>${price}</p></div>`;
};

// Filter with Array.filter()
const filterProperties = () => {
  const suburb = document.querySelector('#suburb-input').value.toLowerCase();
  const filtered = properties.filter(p => p.suburb.toLowerCase().includes(suburb));
  document.querySelector('#grid').innerHTML = filtered.map(createCard).join('');
};

// Event listener
document.querySelector('#suburb-input')
        .addEventListener('input', filterProperties);
Section 6.4 · Async/Await

Asynchronous JavaScript — fetch() API

Modern web applications constantly communicate with backend APIs. JavaScript handles this through Promises and the async/await syntax. The fetch() function makes HTTP requests. async/await (ES2017) makes asynchronous code readable and maintainable (Flanagan 2020, p. 375).

api.jsJavaScript ES2017
async function fetchProperties(suburb, type) {
  try {
    const params   = new URLSearchParams({ suburb, type });
    const response = await fetch(`/api/properties?${params}`, {
      headers: {
        'Authorization': `Bearer ${localStorage.getItem('token')}`,
        'Content-Type':  'application/json'
      }
    });

    if (!response.ok) throw new Error(`HTTP error: ${response.status}`);

    const data = await response.json();
    return data.properties;

  } catch (error) {
    console.error('Failed:', error);
    return [];
  } finally {
    document.querySelector('#loading').classList.add('d-none');
  }
}
