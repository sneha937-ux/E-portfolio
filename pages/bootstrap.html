Section 5.1 · Definition

What is Bootstrap?

Bootstrap is the world's most widely used open-source CSS framework for building responsive, mobile-first web applications. Originally developed at Twitter by Mark Otto and Jacob Thornton in 2010, Bootstrap provides a comprehensive set of pre-built CSS styles, JavaScript components, and a powerful 12-column grid system (Otto & Thornton 2023). According to W3Techs (2024), Bootstrap is used by approximately 19% of all websites globally.

Section 5.3 · Grid System

Bootstrap Grid System

The Bootstrap grid system is built on CSS Flexbox and divides the viewport into 12 equal columns. Developers assign column widths using class names such as col-md-4 (4 of 12 columns = one third width on medium screens). The system is mobile-first: columns stack vertically on small screens by default (Bootstrap Documentation 2024).

grid-example.htmlHTML + Bootstrap
<!-- Responsive 3-column property listing -->
<div class="container">
  <div class="row g-4">

    <!-- Full width mobile, half tablet, third desktop -->
    <div class="col-12 col-md-6 col-lg-4">
      <div class="card h-100 shadow-sm">
        <div class="card-body">
          <h5 class="card-title">12 Harbour View Rd</h5>

        </div>
        <div class="card-footer text-muted small">
