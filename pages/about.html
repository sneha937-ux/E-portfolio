About

This ePortfolio was created as part of the CET138 Full Stack Development module at Kent Institute Australia. It represents Assignment 1, contributing 30% of the overall module mark. The portfolio demonstrates knowledge and practical skills across five core areas: Full Stack Development, HTML, CSS, Bootstrap, and JavaScript.

Assessment Brief Requirements:

  • Demonstrate understanding of technologies used to develop a dynamic web application
  • Design and code static and dynamic web applications
  • Create a portfolio showcasing understanding of all five specified topics
  • Provide working examples or links with explanations for each section
