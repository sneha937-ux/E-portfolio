Section 3.1 · Definition

What is HTML?

HTML (HyperText Markup Language) is the standard language used to create and structure content on the web. It was first proposed by Tim Berners-Lee in 1991 and has since evolved through multiple versions, with HTML5 being the current standard (W3C 2014). HTML defines the structure and meaning of web content using a system of nested elements represented by tags. It is not a programming language — it does not perform logic — but it is the essential skeleton upon which all web pages are built (Duckett 2011, p. 2).

Section 3.3 · Semantic HTML5

Semantic HTML5

Semantic HTML5 introduced elements that carry meaning about their role in the page structure. Prior to HTML5, developers used generic <div> elements for layout. Semantic elements improve accessibility (screen readers can interpret them), improve SEO, and make code more maintainable (Pilgrim 2010, p. 21).

Complete Semantic HTML5 Page Example

index.htmlHTML5
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <title>Aussie Real Estate | Home</title>
  <link rel="stylesheet" href="styles.css" />
</head>
<body>
  <header>
    <nav>
      <a href="/">Home</a>
      <a href="/search">Search Properties</a>
    </nav>
    <h1>Find Your Perfect Australian Property</h1>
  </header>

  <main>
    <section id="search">
      <h2>Property Search</h2>
      <form action="/results" method="GET">
        <label for="suburb">Suburb or Postcode:</label>
        <input type="text" id="suburb" name="suburb"
               placeholder="e.g. Bondi Beach" required />
        <button type="submit">Search</button>
      </form>
    </section>
  </main>

  <footer>
    <p>&copy; 2026 Aussie Real Estate.</p>
  </footer>
</body>
</html>

Source: W3C (2014); Duckett (2011, p. 2)
