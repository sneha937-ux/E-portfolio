Section 4.1 · Definition

What is CSS?

CSS (Cascading Style Sheets) is the language used to describe the visual presentation of an HTML document. While HTML defines structure, CSS defines appearance — including layout, colours, typography, spacing, and responsive behaviour. CSS was first proposed by Håkon Wium Lie in 1994 and standardised by W3C (Meyer & Weyl 2017, p. 4). The word 'cascading' refers to the priority hierarchy used to resolve conflicts when multiple rules apply to the same element: specificity, origin, and order.

CSS Selectors, Properties, and Values

style.cssCSS
/* 1. Element selector */
p {
  font-family: 'Calibri', sans-serif;
  font-size:   16px;
  line-height: 1.6;
  color:       #1E293B;
}

/* 2. Class selector */
.card {
  background-color: #FFFFFF;
  border:           1px solid #E2E8F0;
  border-radius:    8px;
  padding:          1.5rem;
  box-shadow:       0 2px 8px rgba(0,0,0,.08);
  transition:       transform 0.2s ease;
}

/* 3. ID selector */
#hero {
  background-image: linear-gradient(135deg, #1F3864 0%, #2E5FA3 100%);
  color:            #FFFFFF;
  padding:          5rem 2rem;
}

/* 4. Pseudo-class — hover state */
.card:hover {
  transform: translateY(-4px);
  box-shadow: 0 8px 24px rgba(0,0,0,.12);
}

/* 5. Media query — responsive mobile */
@media (max-width: 768px) {
  .card-grid { grid-template-columns: 1fr; }
}

Box Model, Flexbox & Grid

CSS Box Model (from inside out):

  ┌────────────────────────────┐  ← margin   (outside space)
  │  ┌──────────────────────┐  │  ← border   (visible edge)
  │  │  ┌────────────────┐  │  │  ← padding  (inside space)
  │  │  │    CONTENT     │  │  │
  │  │  └────────────────┘  │  │
  │  └──────────────────────┘  │
  └────────────────────────────┘

Source: Meyer & Weyl (2017, p. 56)

CSS Flexbox

CSS
.card-container {
  display:         flex;
  flex-wrap:       wrap;
  gap:             1.5rem;
  justify-content: space-between;
}
.card {
  flex: 1 1 300px;
  max-width: 380px;
}

CSS Grid

CSS
.property-grid {
  display: grid;
  grid-template-columns:
    repeat(auto-fill,
    minmax(300px, 1fr));
  gap: 2rem;
  padding: 2rem;
}

CSS Custom Properties (Variables)

style.cssCSS
/* Define a design system with CSS variables */
:root {
  --colour-primary:    #2E5FA3;
  --colour-secondary:  #1F3864;
  --colour-accent:     #B45309;
  --colour-background: #F1F5F9;
  --colour-text:       #1E293B;
  --font-body:         'Calibri', sans-serif;
  --radius:            8px;
  --shadow:            0 2px 8px rgba(0,0,0,.08);
}

/* Use the variables throughout the stylesheet */
.hero {
  background-color: var(--colour-secondary);
  color:            var(--colour-background);
  border-radius:    var(--radius);
}
